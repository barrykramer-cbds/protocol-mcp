# PROBABILITY INDEX (PI)
## Rapid Bayesian Interpretation Scoring
### Origin: Barry Kramer + Claude | April 14, 2026

---

## THE PROTOCOL

### Phase 1: ENUMERATE INTERPRETATIONS

List every plausible interpretation. There are always at least two.

### Phase 2: SCORE EACH INTERPRETATION

Three dimensions (0-10 each, max 30): Observable Evidence Fit, Prior Frequency, Structural Plausibility.

### Phase 3: COMPARE AND DECIDE

Gap 8+: clear winner. Gap 4-7: proceed but flag. Gap 0-3: find the decisive data point.

### Phase 4: STATE THE IMPLICATION

Translate probability assessment into action or reframing.
