# SIDEBAR PROTOCOL
## Context-Isolated Tangent Framework
### Origin: Barry Kramer + Claude

---

## THE PROTOCOL

### Phase 1: INVOCATION

Trigger: `-sidebar` (optionally followed by a topic label). Mark current position, create isolated context.

### Phase 2: SIDEBAR EXECUTION

Full multi-turn conversation within the sidebar. Topic can evolve naturally.

### Phase 3: EXIT

Triggers: `-main`, `-endsidebar`, or resuming main topic. Main thread resumes from marked position.

### Phase 4: EXPORT (optional)

Trigger: `-sidebar save`. Export sidebar content as clean markdown.
