# DEVIL'S ADVOCATE PROTOCOL (DAP)
## Adversarial Audit Framework
### Origin: Barry Kramer + Claude

---

## THE PROTOCOL

### Phase 1: FRAME THE TARGET

State the proposition as a concrete, falsifiable claim.

### Phase 2: ENUMERATE FACTORS

List every factor affecting success or failure. Every assumption is a factor.

### Phase 3: SCORE EACH FACTOR

Score +5 (strong positive) to -5 (fatal flaw) on a symmetric scale.

### Phase 4: PRODUCE THE SCORECARD

Table sorted worst-first. Include net score and critical factor count (-3 or below).

### Phase 5: HONEST INTERPRETATION

No softening. No motivational close. No hedging negatives.

### Phase 6: REMEDIATION MAP (if applicable)

For each factor at -3 or below: can it be fixed, what would fix it, what does the fix cost.
